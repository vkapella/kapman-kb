# KapMan Knowledge Base

KapMan KB is the versioned trading-rules knowledge base for the KapMan
systematic options framework.

- llm_runtime/ contains runtime rules that are uploaded to LLM project knowledge.
- engineering_only/ contains reference material for humans and engineering tools and is not uploaded to LLMs.

See INDEX.md for the current file directory and v2.3 to v3.0 migration table.
See AGENTS.md for repository working rules.
The active KB version is tracked in CHANGELOG.md.
